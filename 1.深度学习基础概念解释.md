# 深度学习基础概念解释

## 1. 神经网络基础概念

### 神经网络是什么？

想象神经网络就像一个人的大脑学习过程：

- **输入层**：眼睛看到的信息（比如图片）
- **隐藏层**：大脑处理信息的中间过程
- **输出层**：大脑做出的判断结果

### 正向传播（Forward Propagation）

**解决的问题：模型如何做预测？**

就像你看到一个苹果图片：

- 输入：苹果图片的像素数据
- 经过各层处理：识别形状、颜色、纹理等特征
- 输出：判断这是"苹果"的概率

在代码中：

```python
outputs = model(inputs)  # 正向传播：输入→输出
```

### 损失函数（Loss Function）

**解决的问题：预测有多不准？**

就像老师批改作业：

- 你的答案：模型预测是"苹果"
- 正确答案：标签确实是"苹果"
- 损失值：衡量预测与真实的差距

### 反向传播（Backward Propagation）

**解决的问题：如何从错误中学习？**

就像你考试做错题后：

1. **找出错误原因**：`loss.backward()` 计算每个参数对错误的"责任"
2. **调整学习方法**：`optimizer.step()` 根据错误程度调整参数

### 完整学习过程类比

想象你在学习识别水果：

**正向传播（预测阶段）**：

- 看到新水果 → 大脑处理 → 猜测是"梨"

**计算损失（评估阶段）**：

- 正确答案是"苹果" → 发现猜错了

**反向传播（学习阶段）**：

- 分析为什么猜错：形状记错了？颜色记错了？
- 调整记忆：下次看到圆形红色水果要想到苹果

**参数更新（改进阶段）**：

- 更新大脑中的"水果特征记忆"

### 代码中的具体流程

```python
# 1. 清空之前的"学习记录"
optimizer.zero_grad()

# 2. 正向传播：做预测
outputs = model(inputs)  # 模型看到数据，给出预测

# 3. 计算损失：评估预测质量
loss = criterion(outputs, labels)  # 对比预测和真实答案

# 4. 反向传播：分析错误原因
loss.backward()  # 计算每个参数对错误的贡献度

# 5. 参数更新：根据错误调整
optimizer.step()  # 改进模型，下次预测更准
```

**简单总结**：

- **正向传播**：模型"看"数据并"猜"答案
- **反向传播**：模型"反思"为什么猜错并"改进"自己

这样循环训练，模型就会越来越聪明！
