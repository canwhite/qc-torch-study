## 2. 深度学习概念的数学解释

### 神经网络的基本数学结构

**一个神经元**：`y = f(w₁x₁ + w₂x₂ + ... + wₙxₙ + b)`

- `x₁, x₂...`：输入特征
- `w₁, w₂...`：权重（重要性）
- `b`：偏置（调整阈值）
- `f`：激活函数（引入非线性）

### 正向传播的数学过程

**逐层计算**：

```
输入层 → 隐藏层1 → 隐藏层2 → 输出层
   x    →   z₁    →   z₂    →   y
```

**数学公式**：

```
z₁ = f(W₁x + b₁)      # 第一层
z₂ = f(W₂z₁ + b₂)     # 第二层
y = W₃z₂ + b₃         # 输出层
```

在代码中：`outputs = model(inputs)` 就是执行这些矩阵运算

### 损失函数的数学意义

**交叉熵损失**（分类问题常用）：

```
loss = -∑[y_true * log(y_pred)]
```

- `y_true`：真实标签（0 或 1）
- `y_pred`：模型预测的概率
- 当预测完全正确时，loss = 0
- 当预测完全错误时，loss = +∞

### 反向传播的核心：链式法则

**梯度计算**：

```
∂loss/∂w = ∂loss/∂y * ∂y/∂z₂ * ∂z₂/∂z₁ * ∂z₁/∂w
```

**具体例子**：
假设我们有简单网络：`y = w₂ * relu(w₁x + b₁) + b₂`

损失对 w₁ 的梯度：

```
∂loss/∂w₁ = ∂loss/∂y * ∂y/∂z * ∂z/∂w₁
          = 2(y - y_true) * w₂ * (如果z>0则1，否则0) * x
```

### 参数更新的数学

**梯度下降**：

```
w_new = w_old - learning_rate * ∂loss/∂w
```

**代码对应**：

```python
loss.backward()        # 计算所有 ∂loss/∂w
optimizer.step()       # 执行 w = w - lr * ∂loss/∂w
```

### 具体数值例子

假设：

- 输入：x = 2.0
- 权重：w = 1.5, b = 0.5
- 真实值：y_true = 4.0
- 学习率：lr = 0.1

**正向传播**：

```
y_pred = w*x + b = 1.5*2 + 0.5 = 3.5
loss = (y_pred - y_true)² = (3.5-4)² = 0.25
```

**反向传播**：

```
∂loss/∂w = 2*(y_pred - y_true)*x = 2*(3.5-4)*2 = -2.0
```

**参数更新**：

```
w_new = 1.5 - 0.1*(-2.0) = 1.7
```

下次预测：`1.7*2 + 0.5 = 3.9`（更接近 4.0 了！）

### 为什么需要反向传播？

**手动计算太复杂**：

- 3 层网络可能有上百万个参数
- 每个参数都要计算梯度
- 链式法则自动完成这个繁重工作

**数学本质**：反向传播就是**自动微分**，让计算机帮我们计算复杂的偏导数。

这样理解，深度学习其实就是用数学方法让计算机学会"思考"！
